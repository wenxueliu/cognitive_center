## 使用模式与应用场景

### 开发工作流集成
1. **代码理解** → Refact/Zoekt → 代码仓分析
2. **智能搜索** → Zoekt → 快速代码定位
3. **文档生成** → AI-doc-gen → 自动化文档维护
4. **知识管理** → DeepWiki → 知识沉淀

### 工具组合策略
- **搜索+理解**: Zoekt + Refact 组合进行深度代码分析
- **生成+维护**: AI-doc-gen + Claude Code 实现文档自动化
- **规划+执行**: task-master + 代码智能体完成复杂开发任务

## 技术特点分析

### 统一特征
- 基于AI的智能分析能力
- 支持大规模代码仓库处理
- 提供MCP接口便于集成
- 开源生态丰富，可定制性强

### 选择建议
- **代码搜索**: 优先Zoekt（成熟、高性能）
- **文档生成**: AI-doc-gen（防文档腐化设计）
- **任务管理**: claude-task-master（专为Claude优化）
